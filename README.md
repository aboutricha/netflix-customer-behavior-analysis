# Netflix Customer Behavior Analysis (EDA)

## 📌 Objective
To analyze user engagement patterns, subscription behavior, genre preferences, and demographic trends.

## 📊 Dataset
Netflix customer dataset (5000 records)

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 🔍 Key Insights
- Engagement levels are similar across subscription tiers
- Account sharing is common (multiple profiles per account)
- Genre preferences vary across age and gender groups
- Extreme watch-hour outliers exist

## 💡 Business Recommendations
- Improve tier differentiation
- Strengthen profile-level personalization
- Use demographic-aware content targeting

## 📈 Skills Demonstrated
EDA | Data Visualization | User Segmentation | Business Insight Generation
